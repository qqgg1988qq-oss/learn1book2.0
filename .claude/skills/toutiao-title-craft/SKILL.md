# Toutiao Title Craft — 今日头条爆款标题生成器

> 基于161条真实爆款标题的深度分析 | 覆盖5大领域 | 10大核心公式 | 12类结构技巧

## 核心铁律（生成前必读）

1. **前13字定生死** — 手机端标题只显示前13-16字，核心钩子必须前置
2. **一标题三技巧** — 爆款标题平均使用3.2种结构技巧组合
3. **数字是第一吸引力** — 46%的爆款标题包含具体数字
4. **情绪>信息** — 先调动情绪，再传递信息
5. **话说一半留一半** — 47.8%的爆款标题使用悬念机制

## 工作流程

### Step 1: 分析文章内容

从用户提供的文章/主题中提取：
- **核心主题**：文章在讲什么
- **目标人群**：谁会读这篇文章
- **核心利益**：读者能得到什么
- **情感基调**：正面/负面/中性？焦虑/希望/愤怒？
- **领域分类**：社会热点 / 情感生活 / 历史文化 / 科技财经 / 健康养生

### Step 2: 确定心理驱动

根据领域选择核心心理触发机制：

| 领域 | 最强心理驱动 | 触发关键词 |
|------|------------|-----------|
| 社会热点 | 愤怒/正义(20%) + 好奇(20%) | 质问、不公、揭秘、最新 |
| 情感生活 | 好奇(37%) + 共鸣(13%) | 为什么、扎心、崩溃、真相 |
| 历史文化 | 好奇(82%) | 揭秘、不为人知、真相、竟 |
| 科技财经 | 贪婪(67%) | 月入、赚钱、加薪、翻倍 |
| 健康养生 | 恐惧(24%) + 好奇(21%) | 千万别、险送命、别吃、远离 |

### Step 3: 选择主公式 + 叠加辅助技巧

**主公式选择**（从10大公式中选1-2个）：

| # | 公式名 | 模板 | 适用场景 |
|---|-------|------|---------|
| 1 | 数字悬念利益 | `[数字]招[动作]，[利益结果]！[权威]` | 健康/职场/副业 |
| 2 | 反常识颠覆 | `[普遍认知]？[否定]，[反常识结论]` | 科普/历史/健康 |
| 3 | 揭秘真相 | `[揭秘][事件]，[不为人知]的[真相]` | 历史/军事/社会 |
| 4 | 故事数字反转 | `[年龄/身份]+[数字]，[事件]+[反转]` | 情感/家庭/故事 |
| 5 | 痛点利益 | `[痛点场景]，[利益承诺/解决方案]` | 健康/职场/育儿 |
| 6 | 冲突反转爽感 | `[压迫]，[反击]，[爽感结局]` | 职场/家庭冲突 |
| 7 | 设问引导 | `[为什么/如何][现象]？[悬念]` | 情感/育儿/科普 |
| 8 | 热点绑定 | `[热点事件]，[利益/悬念]` | 科技/财经/社会 |
| 9 | 权威揭秘 | `[身份/数据]，[揭秘]+[内容]` | 健康/职场/财经 |
| 10 | 极端情绪 | `[极端情绪/行为]，[反差结果]` | 情感冲突/正能量 |

**辅助技巧叠加**（选1-2个）：
- +数字 → 增强可信度
- +悬念 → 用"它""这一步""这样"不透露答案
- +权威 → 老中医/10年HR/央行数据
- +热点 → 绑定DeepSeek/AI/最新政策
- +冲突 → 制造反差和矛盾
- +情感 → 扎心/泪崩/暖心

### Step 4: 生成候选标题

**输出要求**：
- 生成 **8-12个** 候选标题
- 覆盖不同公式和技巧组合
- 每个标题标注使用的公式编号和技巧标签
- 字数控制在 **16-25字**（最佳区间）
- 前13字必须包含至少1个钩子

**标题格式**：
```
候选1: [标题内容] | 公式X+技巧Y+技巧Z | 预估点击率:高/中
```

### Step 5: 自检与优化

对每条标题进行以下检查：

| 检查项 | 合格标准 |
|--------|---------|
| 字数 | 16-25字最佳，12-28字可接受 |
| 前13字 | 包含数字/悬念/冲突/利益之一 |
| 数字 | 有具体数字优先（3-7最佳） |
| 悬念 | 话说一半，有信息缺口 |
| 情绪 | 能调动至少一种情绪 |
| 利益 | 读者知道"看了能得到什么" |
| 合规 | 不涉及违法违规、过度夸大 |

### Step 6: 推荐标签（可选）

当用户需要为文章添加头条号标签/话题时，执行此步骤。

**标签推荐原则**：
- 最多推荐 **5个标签**，每个标签 **2-8字** 最佳
- 标签必须与文章内容高度相关，避免泛泛而谈
- 优先选择有搜索热度的精准标签

**标签分类体系**：

| 类型 | 数量 | 作用 | 示例 |
|------|------|------|------|
| 核心标签 | 1-2个 | 精准概括文章主题 | #天文考古学 #古代天文仪器 |
| 扩展标签 | 1-2个 | 相关领域延伸 | #中国传统文化 #历史揭秘 |
| 热门标签 | 1个 | 蹭热度增加曝光 | #冷知识 #颠覆认知 |

**领域标签库**：

| 领域 | 推荐标签方向 |
|------|-------------|
| 社会热点 | #热点追踪 #社会百态 #民生关注 #正能量 |
| 情感生活 | #情感共鸣 #家庭生活 #婚姻关系 #人生感悟 |
| 历史文化 | #历史揭秘 #传统文化 #考古发现 #国学智慧 #文物故事 |
| 科技财经 | #科技前沿 #理财知识 #创业经验 #人工智能 |
| 健康养生 | #健康知识 #养生秘诀 #食疗方 #运动健身 |

**标签输出格式**：
```
推荐标签：
  核心：#标签1 #标签2
  扩展：#标签3 #标签4
  热门：#标签5
```

**标签自检**：
- [ ] 标签与文章内容直接相关，不牵强附会
- [ ] 避免过于宽泛的标签（如"#生活""#日常"）
- [ ] 不重复使用意思相近的标签
- [ ] 不添加违法违规、敏感话题标签
