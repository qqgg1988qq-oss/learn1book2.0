## 生成检查清单

### 结构检查

- [ ] 开头钩子能在 30 秒内抓住注意力
- [ ] 每节有过渡句，不生硬跳转
- [ ] 有呼吸点（每 200-300 字一个停顿）
- [ ] 结尾有互动引导或真实感受

### 人味检查

- [ ] 有"我"的观点或反应（每节至少一处）
- [ ] 有具体细节（数字、场景、时间、页码）
- [ ] 有节奏变化（长短句交错）
- [ ] 有承认复杂或不确定的地方
- [ ] 有生活连接（个人经历或类比）

### AI 模式规避检查

- [ ] 无夸大意义（标志着/证明/反映更广泛的）
- [ ] 无知名度堆砌（媒体名称/粉丝数）
- [ ] 无 -ing 肤浅分析（象征着/反映了）
- [ ] 无宣传语言（充满活力的/令人叹为观止的）
- [ ] 无模糊归因（专家指出/业内人士认为）
- [ ] 无挑战-展望套路
- [ ] 无 AI 高频词（此外/至关重要/深入探讨/关键/格局）
- [ ] 无系动词回避（作为/拥有 → 改用 是/有）
- [ ] 无否定排比（不仅……而且……）
- [ ] 无三段式（强行凑三项）
- [ ] 无同义词循环
- [ ] 无虚假范围（从 X 到 Y）
- [ ] 无破折号滥用（≤2 处）
- [ ] 无协作残留（希望有帮助/请告诉我）
- [ ] 无免责声明（截至/基于现有信息）
- [ ] 无谄媚语气（好问题/你说得对）
- [ ] 无填充短语（值得注意的是/在这个时间点）
- [ ] 无过度限定（可能潜在地被认为）
- [ ] 无乐观废话结尾（未来光明/激动人心的时代）
