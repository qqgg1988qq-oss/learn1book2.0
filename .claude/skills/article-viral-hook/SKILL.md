# Article Viral Hook

将长文章转化为高传播短帖，为原文精准引流。

## 工作流程

1. **读文章** — 通读全文，标记核心论点、数据、金句、故事
2. **筛爆点** — 按五维爆点标准筛选 2-4 个最具传播力的观点
3. **选钩子** — 判断每个爆点适合哪种引流钩子类型
4. **写帖文** — 将爆点转化为生活案例帖，植入选定的引流钩子
5. **出结果** — 输出 2-3 篇不同角度的短帖，供用户选择

## 执行细节

### 1. 读文章阶段

通读全文，在思考中记录：
- 作者在解决什么问题？
- 文章最核心的 3 个论点是什么？
- 有哪些反直觉的结论、惊人的数据、戳心的故事？
- 哪些内容如果单独拎出来，会让人想转发/评论/收藏？

### 2. 筛爆点阶段 — 五维爆点标准

一个观点具备以下任一特征即可视为爆点：

| 维度 | 判断标准 | 传播心理 |
|------|---------|---------|
| **反直觉** | 与常识相反，读者第一反应是"不可能吧" | 认知冲突驱动点击 |
| **情绪共鸣** | 戳中普遍存在的焦虑、委屈、渴望 | 情绪需要出口 |
| **身份认同** | 让某个群体觉得"这就是我" | 自我表达需求 |
| **实用利益** | 读完能省时间/省钱/避坑/提高效率 | 功利性收藏转发 |
| **争议空间** | 存在合理对立面，评论区能吵起来 | 讨论欲驱动传播 |

筛选原则：优先选择同时具备 2+ 个维度的观点，或单一维度极强的观点。

### 3. 选钩子阶段

根据爆点特性匹配合适的引流钩子：

| 钩子类型 | 适用场景 | 示例 |
|---------|---------|------|
| **悬念式** | 反直觉观点、有惊人结论 | "看完这篇文章，我把家里所有的XX都扔了——" |
| **对比式** | 打破常见误区、揭示真相 | "你以为的XX，和实际的XX，差了十万八千里" |
| **金句式** | 文章中有精炼的概括性表达 | "有句话最近很火：'XXX'。但很多人不知道后半句" |
| **追问式** | 引发好奇、制造信息差 | "为什么聪明人都在偷偷做这件事？" |
| **故事式** | 文章中有具体案例/数据 | "一个普通XX，用了这个方法后，结果让所有人大跌眼镜" |
| **痛点式** | 实用利益类观点 | "如果你也在为XX烦恼，这篇文章能帮你省掉3年时间" |

### 4. 写帖文阶段

**生活案例化方法**：
- 把文章中的抽象概念，替换成买菜、通勤、做饭、带娃、打工等日常场景
- 用一个具体的小事讲清楚一个大道理
- 让没读过原文的人也能看懂，但读完会产生"原文一定更精彩"的期待

**帖文结构（300-800字）**：
```
[钩子开头] 1-2句，必须抓眼球
↓
[生活案例] 日常场景代入，建立共鸣
↓
[核心观点] 点出爆点，制造认知冲击或情绪高峰
↓
[阅读原文] 自然引导："这篇文章把这个问题讲透了" / "完整的逻辑在原文里"
↓
[互动收尾] 一句话引导评论/转发，如"你遇到过这种情况吗？"
```

**写作原则**：
- 帖文本身要有完整阅读价值，不能是纯粹的广告
- 不要剧透原文全部内容，保留 30-40% 的核心信息在原文中
- 用口语化表达，像朋友聊天，不是论文摘要
- 善用短句、换行、emoji（适度），提升可读性
- 每个观点最多配 1-2 个生活案例，不要堆砌

### 5. 出结果阶段

每次输出 2-3 篇不同角度的短帖：
- 帖文 A：走情绪共鸣路线（适合朋友圈/小红书）
- 帖文 B：走反认知/实用路线（适合知乎/公众号）
- 帖文 C（可选）：走争议讨论路线（适合微博/即刻）

每篇帖文标注：
- 适用平台
- 选择的爆点维度
- 钩子类型

## 输出格式

```markdown
## 文章分析
- 核心主题：一句话概括
- 筛选出的爆点观点（2-4个）：
  1. [观点] — 维度：反直觉/情绪共鸣/...
  2. [观点] — 维度：...

---

## 短帖 A（[路线名]｜适合[平台]）
**爆点：** [选用的观点]
**钩子：** [钩子类型]

[帖文正文]
---

## 短帖 B（[路线名]｜适合[平台]）
...
```

## 质量自检清单

输出前检查每篇帖文：
- [ ] 没读过原文的人能看懂吗？
- [ ] 读完后有想点击原文的冲动吗？
- [ ] 生活案例是否真实可信，不生硬？
- [ ] 有没有制造适当的"信息缺口"（想知道更多→点原文）？
- [ ] 语气是否像真人写的，不是AI/营销号？
