```
{{短文内容}}
```

## 分析框架

### 1. 读短文
- 主题：______
- 情绪：______
- 意象：______
- 隐喻：______

### 2. 定风格
- 短文气质：______
- 视觉风格：______
- 主色调：______
- 光线类型：______

### 3. 设场景
- 场景描述：______
- 设计方法：情绪映射 / 物品借喻 / 动作定格 / 环境氛围 / 视角转换
- 视觉焦点：______

### 4. 写提示词

```
写实风格照片，{{主体}}，{{环境}}，{{光线}}，{{色彩}}，{{氛围}}，{{构图}}，超高清细节，8K
```

### 5. 质检
- [ ] 无文字/字母/数字/符号元素
- [ ] 写实照片级，非插画/油画/动漫
- [ ] 日常生活场景，不过度戏剧化
- [ ] 色彩光线与情绪一致
- [ ] 长度150-350字
