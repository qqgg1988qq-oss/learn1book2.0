# 示例：短文配图

## 输入

> "周末的午后，阳光透过百叶窗洒在木地板上，猫蜷缩在沙发角落打着盹。我捧着一本书，却一个字也没看进去。窗外的梧桐叶沙沙作响，时间仿佛变得很慢很慢。这样的时刻，什么都不想做，只想被安静包围。"

---

## 输出

## 短文分析
- **主题**：周末午后的慵懒与宁静
- **情绪基调**：温暖、松弛、被安静包裹的惬意
- **核心意象**：阳光、猫、百叶窗、木地板、梧桐叶

## 场景设计方案
- **场景描述**：午后的客厅里，一束阳光透过百叶窗在地板上投下条纹光影，一只橘猫蜷在沙发角落熟睡，旁边放着一本翻开的书
- **设计思路**：用"阳光条纹+熟睡的猫+翻开的书"组合，传递无所事事却无比舒适的周末午后氛围。不出现人物，让观者有代入感

## 生图提示词（中文）

```
写实风格照片，一间温馨的客厅局部特写，浅棕色木地板上投射着百叶窗切割出的条状光影，一只橘色虎斑猫蜷缩在米色布艺沙发的角落里熟睡，沙发扶手上放着一本翻开的书页微微卷曲的无字书籍，背景是一扇木质百叶窗，窗外隐约可见梧桐树叶的绿色轮廓，午后柔和的金色阳光从百叶窗缝隙倾泻而入，空气中可见细微的尘埃在光线中浮动，整体色调偏暖黄和浅棕，氛围慵懒而宁静，仿佛时间停止在周末下午三点，中景构图，焦点在猫咪蜷缩的身体和地板的光影条纹上，背景适度虚化，超高清细节，8K
```

## 备选方案

**场景描述**：从猫咪视角拍摄的仰视画面，窗外梧桐叶在微风中摇曳，阳光在叶片间闪烁

```
写实风格照片，一只橘猫的背部和耳朵从画面下方入镜，视角微微仰视，背景是一扇大窗户，窗外翠绿的梧桐树叶在微风中轻轻摇曳，午后的阳光透过叶片的缝隙洒入室内，在猫毛上形成斑驳的光点，窗台上放着一个无字的陶瓷马克杯，整体色调偏清新绿和暖金色，氛围安逸而充满生机，仿佛能感受到微风和阳光的触感，浅景深构图，焦点在猫耳和窗外的叶片上，超高清细节，8K
```
