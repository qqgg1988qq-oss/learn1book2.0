# Text to Image Prompt

将短文本转化为写实风格的中文生图提示词，用视觉场景表达文字的情绪与意境。

## 工作流程

1. **读短文** — 理解主题、情绪基调、核心意象
2. **定风格** — 根据短文气质确定视觉风格与色调
3. **设场景** — 设计一个日常生活场景来呼应短文
4. **写提示词** — 用中文撰写详细的写实风格生图提示词
5. **质检** — 确保无文字元素，风格统一，贴合文意

## 执行细节

### 1. 读短文阶段

在思考中分析：
- **主题**：这段文字在讲什么？核心话题是什么？
- **情绪**：整体基调是温暖、孤独、焦虑、希望、怀旧、轻松……？
- **意象**：文字中有哪些可以被视觉化的元素？（如：咖啡、雨夜、窗台、旧书、路灯等）
- **隐喻**：文字在表达什么深层感受？需要用什么场景来"暗示"这种感受？

### 2. 定风格阶段

根据短文气质选择视觉方向：

| 短文气质 | 视觉风格 | 色调 | 光线 |
|---------|---------|------|------|
| 温暖治愈 | 日系生活风 | 暖黄、米色、淡棕 | 柔和自然光 |
| 孤独安静 | 清冷日常风 | 灰蓝、浅灰、暗绿 | 散射光/暮色 |
| 焦虑迷茫 | 都市疏离风 | 霓虹冷色、暗调 | 人工光源对比 |
| 希望向上 | 明亮清新风 | 白、浅蓝、嫩绿 | 晨光/阳光充足 |
| 怀旧复古 | 胶片感日常 | 褪色暖色、棕调 | 侧逆光/黄昏 |
| 轻松愉悦 | 明快生活风 | 鲜艳但不刺眼 | 充足漫射光 |

**关键**：写实风格（photorealistic），不是插画、不是动漫、不是油画。照片级真实感。

### 3. 设场景阶段 — 场景设计方法论

**核心原则**：场景不是文字的"直接图解"，而是文字的"视觉共鸣"。用日常生活中一个具体的、有氛围的瞬间，来呼应短文表达的情绪或主题。

**场景设计方法**：

- **情绪映射法**：文字讲孤独 → 一个空荡的地铁站台，一个人等车的背影
- **物品借喻法**：文字讲思念 → 一杯冒着热气的茶，旁边放着未拆的信
- **动作定格法**：文字讲坚持 → 清晨六点半，厨房台面上正在煮的咖啡
- **环境氛围法**：文字讲迷茫 → 雨后模糊的玻璃窗，窗外霓虹灯光晕染
- **视角转换法**：文字讲成长 → 从小孩子的视角仰望大人的书架

**场景选择约束**：
- 必须是日常生活中真实存在的场景
- 避免过于戏剧化、电影化的夸张构图
- 优先选择安静、内敛、有留白的画面
- 场景中可以有1-2个视觉焦点，但不要杂乱

### 4. 写提示词阶段

**输出一条完整的中文提示词**，结构如下：

```
写实风格照片，[主体描述]，[环境/背景描述]，[光线描述]，[色彩描述]，[氛围/情绪描述]，[构图/视角描述]，超高清细节，8K
```

**各模块撰写要求**：

| 模块 | 内容 | 示例 |
|------|------|------|
| 主体 | 画面中核心的人/物/动作 | 一个年轻女子坐在窗边的旧沙发上 |
| 环境 | 场景设置、背景元素 | 窗外是细雨蒙蒙的城市街景，窗台上放着一盆绿植和一本翻开的书 |
| 光线 | 光源方向、强度、质感 | 柔和的散射自然光从窗外照进来，在书页上形成淡淡的光斑 |
| 色彩 | 主色调、配色方案 | 整体色调偏暖灰和淡绿，温馨而安静 |
| 氛围 | 情绪感、故事感 | 氛围宁静而略带思绪，仿佛时间在这一刻放慢 |
| 构图 | 视角、景别、焦点 | 中景构图，焦点在人物侧脸和书页上，背景适度虚化 |

**提示词原则**：
- 全程中文，不要用英文
- 描述要具体但不做作，像在给摄影师下 brief
- 明确指定"写实风格照片"、"超高清细节"、"8K"
- **绝对不要包含任何文字、字母、符号类元素**（如招牌、书本上的字、屏幕文字等）
- 如果要出现书本、手机、招牌等可能带字的物体，明确描述为"无字的"、"空白屏幕"

### 5. 质检阶段

输出前逐项检查：
- [ ] 提示词中没有任何文字、字母、数字、符号类描述
- [ ] 风格为写实照片级，不是插画/油画/动漫
- [ ] 场景来自日常生活，不过度戏剧化
- [ ] 色彩和光线描述与短文情绪一致
- [ ] 提示词长度适中（150-350字），信息密度高

## 输出格式

```markdown
## 短文分析
- **主题**：[一句话概括]
- **情绪基调**：[如：温暖中带点孤独]
- **核心意象**：[2-3个关键词]

## 场景设计方案
- **场景描述**：[用1-2句话描述画面]
- **设计思路**：[为什么选择这个场景来呼应短文]

## 生图提示词（中文）

```
[完整的中文提示词，一行或多行均可]
```

## 备选方案（可选）
如果短文适合多种视觉表达，提供一个备选场景和提示词。
```

## 示例

**输入短文**：
> "有时候觉得，成长就是把哭声调成静音的过程。以前难过会找人倾诉，现在只是默默地泡一杯茶，坐在窗边发呆。不是不想说，是说了也没用，不如自己消化。"

**输出**：

## 短文分析
- **主题**：成长的孤独与自我消化
- **情绪基调**：安静、克制、淡淡的怅然
- **核心意象**：茶、窗边、沉默、独处

## 场景设计方案
- **场景描述**：窗边的木质小桌上，一杯冒着淡淡热气的茶，窗外是模糊的城市夜景
- **设计思路**：用"一杯热茶+空椅子+窗外夜色"的组合，暗示一个人默默消化情绪的瞬间，不出现人物本身，用空寂感传递情绪

## 生图提示词（中文）

```
写实风格照片，一张木质小圆桌的特写，桌面上放着一只白瓷茶杯，杯中茶水冒着淡淡的热气，旁边是一个空着的旧藤椅靠背，背景是一扇老式木框玻璃窗，窗外是模糊的城市夜景灯光，暖黄色的室内灯光与窗外冷色调形成对比，整体色调偏暖棕和暗蓝，氛围安静而略带寂寥，仿佛刚刚有人离开，中景构图，焦点在茶杯的热气和桌面纹理上，背景适度虚化，超高清细节，8K
```
