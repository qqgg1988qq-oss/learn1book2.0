### 交互行为

1. **悬浮概念**：鼠标放在"附随性"上 → 显示 tooltip"就像 LED 显示屏上的图像和像素的关系..."
2. **点击概念**：点击"附随性" → 右侧概念卡片高亮并滚动到视野中央，展开完整定义
3. **滚动原文**：滚动到讨论"容易问题"的段落 → 右侧"容易问题"卡片自动高亮
4. **展开卡片**：点击右侧"困难问题"卡片 → 展开显示：术语名称、定义、解释、通俗解释、关联概念

## 生成的 HTML 特性

- **零依赖**：单文件 HTML，CSS/JS/数据全部内联
- **暗色模式**：自动跟随系统偏好
- **移动端适配**：窄屏自动隐藏右侧，变为上下布局
- **进度记忆**：LocalStorage 记录阅读进度（可选）
