# 叙事转解说文案 — 处理模板

## 输入

```
原文内容：[粘贴叙事性文章全文]

可选要求：
- 时长：[如：8分钟 / 10分钟 / 不限制]
- 风格：[悬疑紧张 / 温情治愈 / 犀利毒舌 / 偏纪录片 / 偏娱乐化]
- 受众：[Z世代 / 文学爱好者 / 历史爱好者 / 泛娱乐观众]
- 特殊要求：[如：侧重人物心理 / 强调时代背景 / 需要古今对照]
```

## 处理流程

### 阶段一：叙事骨架提取（内部思考，不输出）

1. **核心冲突定位**：这个故事的根本矛盾是什么？（人与命运/人与社会/人与自我/人与人）
2. **叙事弧线提取**：
   - 起：初始状态与入局契机
   - 承：欲望升级与阻力叠加
   - 转：不可逆转的转折与代价
   - 合：结局与余韵（开放式或闭环式）
3. **人物动机链**：每个关键行为背后的"为什么"必须可追溯
4. **时间锚点**：标记决定故事走向的"关键时间节点"或"因果触发点"

> 约束：即使改变叙述顺序（倒叙、插叙、多线并置），以上4个要素必须完整保留。

### 阶段二：叙事重构（输出层）

- **因果闭环**：任何情节推进都必须让观众理解"为什么会发生"
- **人物一致性**：核心性格与决策逻辑不得扭曲
- **情感留痕**：强烈情感共鸣场景必须保留并放大
- **取舍原则**：可精简次要支线，但主线"起-承-转-合"节点一个不能少

### 阶段三：讲解语言转换

- **开场钩子**：以悬念、反常识判断、强烈冲突或核心问题开场
- **节奏控制**：每2-3分钟一个"情绪小高潮"或"认知刷新点"
- **画面感营造**：将内心/环境描写转化为"可视觉化"的讲解语言
- **过渡衔接**：段落间用承上启下的逻辑连接词或设问句

### 阶段四：独特评价注入（至少覆盖3个维度）

| 维度 | 说明 |
|------|------|
| 主题解读 | 表层剧情之下隐藏的核心命题 |
| 人物心理深描 | 反常行为背后的心理机制 |
| 时代/环境隐喻 | 背景如何参与人物命运的书写 |
| 价值重构 | 原文的"善恶/成败/得失"是否需要重新审视 |
| 情感共鸣锚点 | 与当代观众普遍经验的链接 |
| 互文与对照 | 关联同类作品或普遍人性母题 |
