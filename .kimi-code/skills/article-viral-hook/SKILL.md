# Article Viral Hook × APAG × 痛点图谱

将长文章转化为高传播短帖，三步工作流：
**痛点匹配 → 爆点筛选 → APAG 帖文生成**

## 工作流程

```
用户提供文章
    ↓
[Step 1] 读文章 + 痛点匹配
    ↓
[Step 2] 筛爆点（五维标准 × 痛点匹配度）
    ↓
[Step 3] APAG 框架写帖（Attention → Perspective → Advantage → Amplify/Gamify）
    ↓
输出 2-3 篇不同角度的短帖
```

## Step 2：筛爆点（五维标准 × 痛点匹配度）

### 五维爆点标准

| 维度 | 判断标准 | 传播心理 |
|------|---------|---------|
| **反直觉** | 与常识相反，读者第一反应是"不可能吧" | 认知冲突驱动点击 |
| **情绪共鸣** | 戳中普遍存在的焦虑、委屈、渴望 | 情绪需要出口 |
| **身份认同** | 让某个群体觉得"这就是我" | 自我表达需求 |
| **实用利益** | 读完能省时间/省钱/避坑/提高效率 | 功利性收藏转发 |
| **争议空间** | 存在合理对立面，评论区能吵起来 | 讨论欲驱动传播 |

### 综合评分规则

- 同时命中 **高概率痛点（>60%）+ 多个爆点维度** → 优先选用
- 单一维度极强（如极强的反直觉数据）→ 第二优先
- 匹配度低但传播性极强 → 可作为辅助帖

## Step 3：APAG 框架写帖（⚠️ 核心写作框架）

**必须使用 APAG 四段式结构**，每篇帖文按以下框架组织：

### APAG 框架速查

```
A — Attention（钩子）     → 占 10-15%，3秒抓眼球
P — Perspective（视角）   → 占 20-30%，描绘读者痛点，让TA感到"这说的就是我"
A — Advantage（优势）     → 占 30-40%，展示新视角+证据，让TA相信有更好的方式
G — Amplify/Gamify（放大）→ 占 15-20%，给出可执行步骤+CTA/互动引导
```

### APAG 各阶段详解（融入痛点数据）

#### A 阶段：Attention（钩子）

从钩子类型库中选择，**优先选择与匹配痛点直接相关的钩子**：

| 钩子类型 | 适用场景 | 痛点锚定示例 |
|---------|---------|------------|
| **问题型** | 痛点共鸣 | "为什么过了35岁，简历就没人看了？"（锚定 #1） |
| **数字型** | 反直觉数据 | "80%的中年人正在经历……"（引用痛点数据） |
| **反常识型** | 打破迷思 | "你以为35岁是天花板？科学说正好相反" |
| **痛点共鸣型** | 情绪入口 | "如果你也因为年龄焦虑失眠……"（锚定 #2） |
| **时间框架型** | 转变叙事 | "30岁那年我以为人生结束了，40岁才发现……" |
| **负面经历型** | 信任建立 | "39岁之前，我投了200份简历没一个回复" |

#### P 阶段：Perspective（描绘问题视角）

**核心任务**：让读者感到"你懂我"——描绘他们当前的困境、错误认知、痛苦。

**必须做到**：
1. 用具体场景描绘痛点（不是抽象概括）
2. 引用痛点图谱中的社会数据作为"你不是一个人"的证据
3. **不要在这个阶段给答案**——先描绘痛苦，再在后文给方案

**写作公式**：
```
[痛点场景化描述] + [社会数据佐证："80%的中年人也在经历这个"] + ["你是不是也……"句式拉入代入]
```

#### A 阶段：Advantage（展示优势方案）

**核心任务**：提出新视角/新概念/新方案，让读者相信存在更好的方式。

**必须包含（至少2项）**：
- [ ] 文章中的核心新观点/新概念
- [ ] 研究/数据支撑（可引用原文证据）
- [ ] 引用/名言/金句
- [ ] 个人经验/案例故事
- [ ] 社会证明（名人案例/统计数据）

**写作公式**：
```
[转折："直到我读到/发现/意识到……"] + [新视角一句话] + [证据支撑] + [这个新视角为什么更对]
```

#### G 阶段：Amplify/Gamify（放大/行动引导）

**核心任务**：把转变拆解为可执行的步骤 + 引导读者行动。

**必须包含**：
1. **1-3 个具体可执行步骤**（不是空洞口号，是今天就能做的事）
2. **一个明确的 CTA**（行动号召）：
   - 互动型："你有没有经历过……？评论区聊聊"
   - 引流型："完整的论证和步骤在原文里"
   - 行动型："今天就试这一件事……"

**写作公式**：
```
[可执行步骤 1-3 条] + [引导看原文/完整内容] + [互动 CTA（提问/评论区邀请/行动指令）]
```

## Step 4：平台适配

| 平台 | 字数 | APAG 配比 | 风格特点 |
|------|------|-----------|----------|
| **微头条** | 300-1500字 | A10% P25% A40% G25% | 可偏长，段落感，可加话题标签 |
| **微博** | 200-2000字 | A15% P25% A35% G25% | 精炼有力，善用超话，互动感强 |
| **小红书** | 300-1000字 | A15% P30% A30% G25% | 情绪先行，emoji适度，生活场景感 |
| **知乎** | 500-2000字 | A10% P20% A45% G25% | 重数据和逻辑，专业但不学术 |
| **X/Twitter** | 280-4000字 | 精炼版APAG | 极简，可thread拆分 |
