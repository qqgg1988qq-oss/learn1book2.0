## Stage 2：韩寒口语化改写检查清单

对每篇帖文检查：

- [ ] 所有学术术语 → 大白话？
- [ ] 长句已拆为短句？
- [ ] 有"说白了/说实话/我觉得"等口语化开头（但不超过3次）？
- [ ] 数据用"说人话"的方式重新包装？
- [ ] 有冷幽默/反讽/自嘲？
- [ ] 结尾留了退路？
- [ ] **没有**韩寒本人的真实经历？
- [ ] **没有**扮演韩寒身份（"我是韩寒"）？
- [ ] **没有**免责声明？
