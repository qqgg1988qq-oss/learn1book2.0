# Article to Han Post — 输入输出示例

## 示例：完整三阶段流水线

### 输入

用户说：
> "帮我把《大器晚成》的整合报告写成韩寒风格的引流帖"

源文件：`/path/to/大器晚成/整合报告.md`

---

### Stage 1 产出（article-viral-hook 初稿片段）

```
## 短帖 A（情绪共鸣｜适合小红书/朋友圈）

我不是来安慰你的。我是来告诉你一个被埋没的科学事实。

你有没有这种感觉——刷朋友圈，同龄人不是总监就是创业成功；
刷招聘网站，满屏"35岁以下"……

[APAG 初稿语气偏正式/学术，包含"前额皮质""髓鞘形成""液态智力"等术语]
```

### Stage 2 产出（韩寒口语化改写后）

```
我二十岁那年，觉得自己这辈子就这样了。

后来发现，每个人二十岁的时候都觉得自己这辈子就这样了。
区别只是有的人三十岁还在这么想，有的人不这么想了。

说实话，我没资格安慰你。但有个事我觉得值得说一下——

人的大脑，二十五岁才刚长全。

管你决策、管你自控的那块地方，叫前额什么皮质的，医学名词我不懂。
说白了就是：你二十岁的时候做的那些蠢事、考砸的试、走错的弯路，
不是你不行，是你脑袋还没装完。
```

**关键区别**：
- "前额皮质"→"管你决策的那块地方，叫前额什么皮质的，医学名词我不懂"
- "被埋没的科学事实"→"有个事我觉得值得说一下"
- 加入自嘲和第一人称感受
- 长句拆为短句
- **没有使用韩寒真实经历**（没提《三重门》、赛车、拍电影）

### Stage 3 产出（text-humanizer-zh 精修后）

移除的问题：
- "说白了"在帖文中出现了4次 → 减少到2次
- 一个段落有3个"——"破折号 → 替换为句号或逗号
- 3个行动项 → 改为2个 + 个人感受

质量评分：45/50

---

## 关键约束示例

### ❌ 错误：使用了韩寒真实经历

> 我写《三重门》的时候也这样，被所有人说"一个辍学生能写出什么东西"。后来卖了四十万本，那些人就闭嘴了。

### ✅ 正确：使用韩寒的表达方式，但用自己的例子

> 被所有人说"你不行"的时候怎么办？说实话，我也没有好办法。但有一点——别人说你不行，和你能不能行，是两件事。

### ❌ 错误：扮演韩寒身份

> 我是韩寒。我看完这篇文章，觉得写得太学院派了。

### ✅ 正确：用韩寒风格说话，但不宣称是韩寒

> 说实话，这篇文章我看完了。写得挺好，就是太像论文了。
