# 文本人性化处理模板

## 输入

**需要处理的文本：**

```
{{PASTE_TEXT_HERE}}
```

**预期语气（可选）：**
- [ ] 正式/学术
- [ ] 随意/口语
- [ ] 专业/商务
- [ ] 文艺/感性
- [ ] 技术/分析
- [ ] 其他：______

**处理深度（可选）：**
- [ ] 轻度 — 仅去除明显的 AI 词汇和填充短语
- [ ] 中度 — 重写公式化结构，注入自然节奏（默认）
- [ ] 深度 — 全面重构，注入个性与观点

## 处理清单

在改写过程中逐项检查：

### 内容模式
- [ ] 1. 过度强调意义/遗产/更广泛趋势
- [ ] 2. 过度强调知名度和媒体报道
- [ ] 3. 以 -ing 结尾的肤浅分析
- [ ] 4. 宣传和广告式语言
- [ ] 5. 模糊归因和含糊措辞
- [ ] 6. 提纲式的"挑战与未来展望"部分

### 语言模式
- [ ] 7. 过度使用的 AI 词汇（此外、至关重要、关键、复杂等）
- [ ] 8. 避免使用"是"（系动词回避）
- [ ] 9. 否定式排比（不仅……而且……）
- [ ] 10. 三段式法则过度使用
- [ ] 11. 刻意换词（同义词循环）
- [ ] 12. 虚假范围（从 X 到 Y）

### 风格模式
- [ ] 13. 破折号过度使用
- [ ] 14. 粗体过度使用
- [ ] 15. 内联标题垂直列表
- [ ] 16. 标题大写问题（英文适用）
- [ ] 17. 表情符号装饰
- [ ] 18. 弯引号问题

### 交流痕迹
- [ ] 19. 协作交流痕迹（希望有帮助、当然！等）
- [ ] 20. 知识截止日期免责声明
- [ ] 21. 谄媚/卑躬屈膝的语气

### 填充与回避
- [ ] 22. 填充短语
- [ ] 23. 过度限定
- [ ] 24. 通用积极结论

### 灵魂检查
- [ ] 句子长度是否变化？
- [ ] 是否有观点或反应？
- [ ] 是否承认复杂性/不确定性？
- [ ] 是否适当使用"我"或第一人称？
- [ ] 是否有具体细节而非模糊主张？
- [ ] 结尾是否多样化（不总是金句）？
