# Text to Image Prompt

将短文本转化为中文生图提示词，用视觉场景表达文字的情绪与意境。

## 工作流程

1. **读短文** — 理解主题、情绪基调、核心意象
2. **选风格** — 向用户展示可用的视觉风格预设，等待用户选择（或根据短文气质推荐）
3. **定色调** — 根据选定风格确定配色与光线
4. **设场景** — 设计一个具体场景来呼应短文
5. **写提示词** — 用中文撰写符合选定风格的详细生图提示词
6. **质检** — 确保无文字元素，风格统一，贴合文意

---

## 执行细节

### 1. 读短文阶段

在思考中分析：
- **主题**：这段文字在讲什么？核心话题是什么？
- **情绪**：整体基调是温暖、孤独、焦虑、希望、怀旧、轻松……？
- **意象**：文字中有哪些可以被视觉化的元素？（如：咖啡、雨夜、窗台、旧书、路灯等）
- **隐喻**：文字在表达什么深层感受？需要用什么场景来"暗示"这种感受？

---

### 2. 选风格阶段

向用户展示可用的视觉风格预设，等待用户选择：

| 序号 | 风格名称 | 风格描述 | 适用场景 |
|------|---------|---------|---------|
| 1 | `photorealistic` | 写实照片风格 | 日常情绪、氛围感、生活瞬间 |
| 2 | `warm-isometric` | 温暖治愈等距3D插画风 | 知识科普、文化内容、教育主题 |
| 3 | `academic-tech` | 学术科技风 | 哲学思辨、学术解读、深度内容 |
| 4 | `literary-warm` | 文学温暖风 | 文学解读、散文、人物故事 |
| 5 | `documentary-grey` | 纪实灰调风 | 历史回顾、严肃主题、社会观察 |
| 6 | `hand-drawn-journal` | 手绘涂鸦手账风 | 轻松知识、课堂笔记、手账风格 |
| 7 | `lofi-chill-vector` | Lo-Fi Chill撞色矢量插画风 | 现代文艺、深夜阅读、哲学沉思 |

用户也可以提供自定义风格描述。

---

### 3. 视觉风格预设

#### `photorealistic` —— 写实照片风格

- **氛围**：真实、日常、有生活气息
- **视觉倾向**：照片级真实感，自然光影，细腻材质
- **典型元素**：真实人物、日常物品、自然光线、城市/室内环境
- **提示词前缀**：`写实风格照片，`

#### `warm-isometric` —— 温暖治愈等距3D插画风

- **氛围**：温馨治愈、知识友好、可爱亲切
- **视觉倾向**：等距透视(2.5D)，扁平化+柔和立体阴影，米色/奶油色/浅棕色暖基调
- **典型元素**：Q版微型人物、微型场景摆件、木牌/标牌、可爱图标
- **提示词前缀**：`温暖治愈等距3D插画，`

#### `academic-tech` —— 学术科技风

- **氛围**：理性、克制、信息密度高
- **视觉倾向**：几何线条和信息图为主，深色底配亮色点缀
- **典型元素**：数据面板、几何图表、符号系统、电子标签
- **提示词前缀**：`学术科技风插画，`

#### `literary-warm` —— 文学温暖风

- **氛围**：温暖、柔和、有人文气息
- **视觉倾向**：暖色调为主，复古纸张质感
- **典型元素**：手写笔触、暖光、插画感画面
- **提示词前缀**：`文学温暖风插画，`

#### `documentary-grey` —— 纪实灰调风

- **氛围**：冷静、客观、有历史厚重感
- **视觉倾向**：黑白灰为主，偶现暖色点缀
- **典型元素**：档案照片、时间轴、文献质感
- **提示词前缀**：`纪实灰调风格图像，`

#### `hand-drawn-journal` —— 手绘涂鸦手账风

- **氛围**：轻松亲切、活泼可爱、像课堂笔记
- **视觉倾向**：米黄色笔记本纸张背景、黑色手绘抖动感轮廓线、鲜艳马克笔填充色
- **典型元素**：简笔画火柴人角色（点状眼睛、弧线嘴巴）、星星、爱心、灯泡、云朵、对话框、便利贴、手写体与印刷体混排的中文文字
- **提示词前缀**：`手绘涂鸦手账风插画，`

#### `lofi-chill-vector` —— Lo-Fi Chill撞色矢量插画风

- **氛围**：松弛、复古未来感、适合深夜阅读和哲学沉思的 Lo-Fi Chill 氛围
- **视觉倾向**：现代数字插画，平面矢量质感，深色清晰描边，色块分明无渐变，灰绿色调几何拼贴背景
- **典型配色**：珊瑚粉 `#FF6B6B`、深蓝紫 `#4A2C8A`、薄荷绿 `#4ECDC4`、米黄 `#FFE66D`
- **典型元素**：几何色块拼贴、复古磁带/黑胶/台灯等 Lo-Fi 小物件、扁平化人物、抽象符号与文字标牌
- **提示词前缀**：`Lo-Fi Chill撞色矢量插画，`

---

### 4. 定色调阶段

根据选定风格确定视觉方向：

| 风格 | 典型色调 | 典型光线 |
|------|---------|---------|
| `photorealistic` | 按短文情绪选择 | 按短文情绪选择 |
| `warm-isometric` | 米色、奶油色、浅棕色暖基调 | 柔和均匀光 |
| `academic-tech` | 深色底配亮色点缀 | 冷色调人工光 |
| `literary-warm` | 暖色调、复古纸张色 | 暖光、侧逆光 |
| `documentary-grey` | 黑白灰为主，偶现暖色 | 自然光、散射光 |
| `hand-drawn-journal` | 米黄纸张+鲜艳马克笔色 | 均匀明亮 |
| `lofi-chill-vector` | 珊瑚粉、深蓝紫、薄荷绿、米黄 | 柔和氛围光 |

**默认规则**：若用户未明确选择风格，根据短文气质推荐一个最合适的风格，并以写实照片风格（`photorealistic`）作为兜底默认。

---

### 5. 设场景阶段 — 场景设计方法论

**核心原则**：场景不是文字的"直接图解"，而是文字的"视觉共鸣"。用一个具体的、有氛围的瞬间，来呼应短文表达的情绪或主题。

**场景设计方法**：

- **情绪映射法**：文字讲孤独 → 一个空荡的地铁站台，一个人等车的背影
- **物品借喻法**：文字讲思念 → 一杯冒着热气的茶，旁边放着未拆的信
- **动作定格法**：文字讲坚持 → 清晨六点半，厨房台面上正在煮的咖啡
- **环境氛围法**：文字讲迷茫 → 雨后模糊的玻璃窗，窗外霓虹灯光晕染
- **视角转换法**：文字讲成长 → 从小孩子的视角仰望大人的书架

**场景选择约束**：
- 优先选择安静、内敛、有留白的画面
- 场景中可以有1-2个视觉焦点，但不要杂乱
- 写实照片风格避免过于戏剧化、电影化的夸张构图

---

### 6. 写提示词阶段

**输出一条完整的中文提示词**，结构如下：

```
[风格前缀]，[主体描述]，[环境/背景描述]，[光线描述]，[色彩描述]，[氛围/情绪描述]，[构图/视角描述]，超高清细节，8K
```

**各模块撰写要求**：

| 模块 | 内容 | 示例 |
|------|------|------|
| 风格前缀 | 根据选定风格添加 | 写实风格照片 / 温暖治愈等距3D插画 |
| 主体 | 画面中核心的人/物/动作 | 一个年轻女子坐在窗边的旧沙发上 |
| 环境 | 场景设置、背景元素 | 窗外是细雨蒙蒙的城市街景，窗台上放着一盆绿植 |
| 光线 | 光源方向、强度、质感 | 柔和的散射自然光从窗外照进来 |
| 色彩 | 主色调、配色方案 | 整体色调偏暖灰和淡绿，温馨而安静 |
| 氛围 | 情绪感、故事感 | 氛围宁静而略带思绪 |
| 构图 | 视角、景别、焦点 | 中景构图，焦点在人物侧脸，背景适度虚化 |

**提示词原则**：
- 全程中文，不要用英文
- 描述要具体但不做作，像在给视觉创作者下 brief
- 明确指定"超高清细节，8K"
- **绝对不要包含任何文字、字母、符号类元素**（如招牌、书本上的字、屏幕文字等）
- 如果要出现书本、手机、招牌等可能带字的物体，明确描述为"无字的"、"空白屏幕"
- 风格特征要明确写入提示词，确保输出符合选定风格

---

### 7. 质检阶段

输出前逐项检查：
- [ ] 提示词中没有任何文字、字母、数字、符号类描述
- [ ] 写实照片风格：画面为照片级真实感，不是插画/油画/动漫
- [ ] 插画/矢量风格：画面符合选定风格的视觉特征
- [ ] 场景贴合短文情绪与主题
- [ ] 色彩和光线描述与选定风格一致
- [ ] 提示词长度适中（150-350字），信息密度高

## 示例

**输入短文**：
> "有时候觉得，成长就是把哭声调成静音的过程。以前难过会找人倾诉，现在只是默默地泡一杯茶，坐在窗边发呆。不是不想说，是说了也没用，不如自己消化。"

**输出**：

## 短文分析
- **主题**：成长的孤独与自我消化
- **情绪基调**：安静、克制、淡淡的怅然
- **核心意象**：茶、窗边、沉默、独处

## 风格选择
- **选定风格**：`photorealistic`（写实照片风格）
- **选择理由**：短文表达的是日常生活中细腻的情绪瞬间，写实照片最能传递真实感和共鸣

## 场景设计方案
- **场景描述**：窗边的木质小桌上，一杯冒着淡淡热气的茶，窗外是模糊的城市夜景
- **设计思路**：用"一杯热茶+空椅子+窗外夜色"的组合，暗示一个人默默消化情绪的瞬间，不出现人物本身，用空寂感传递情绪
