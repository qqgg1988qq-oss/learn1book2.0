# xl-card 输出模板

```
║  📖 长文卡                           ║
╠══════════════════════════════════════╣
║                                      ║
║  【主标题】                            ║
║  {{主标题，≤20字}}                    ║
║                                      ║
║  【副标题】                            ║
║  {{副标题，≤30字，可选}}               ║
║                                      ║
║  【金句/高亮】                          ║
║  ▎{{金句文案}}                        ║
║                                      ║
║  【正文】                              ║
║  {{段落1}}                            ║
║  {{段落2}}                            ║
║  {{段落3}}                            ║
║                                      ║
║  【条目组】                            ║
║  ● {{条目标题1}}：{{条目正文}}          ║

║                                      ║
║  【行动提示】                           ║

║  八零后阿力的觉醒日记  ·  {{来源}}      ║
```


## 多卡（-m）

每张卡使用上述长文卡模板，按以下格式输出：

```
## 卡 1/N：{{卡片主题}}

[长文卡格式]

---

## 卡 2/N：{{卡片主题}}

[长文卡格式]

---

...
```
