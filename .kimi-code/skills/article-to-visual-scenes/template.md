# 输入模板

将文章/口播文案粘贴到下方对应区域，然后调用 `/article-to-visual-scenes`。

---

## 基础调用

```
/article-to-visual-scenes

[文章正文粘贴在这里]
```

---

## 带参数调用

```
/article-to-visual-scenes

风格预设: literary-warm
输出路径: ./output/my-video-scenes.json
目标平台: bilibili, youtube
预估时长: 18 分钟

[文章正文粘贴在这里]
```

---

## 风格自定义调用

```
/article-to-visual-scenes

风格描述: 我希望视频整体偏复古胶片质感，色调偏棕黄，节奏缓慢，
适合讲述一个怀旧的历史故事。配乐用钢琴和弦乐。

[文章正文粘贴在这里]
```

---

## 强制覆盖风格字段

```
/article-to-visual-scenes

风格预设: academic-tech
配色覆盖:
  primary: "#2C3E50"
  accent: "#E74C3C"
动画风格覆盖: 增加图表的弹跳进入效果

[文章正文粘贴在这里]
```

---

## 输出选项

可选指定输出方式：

- **保存到文件**：提供 `输出路径` 参数（建议 `.json` 后缀）
- **返回到对话**：不提供输出路径，直接返回 JSON 内容
- **同时输出**：提供 `输出路径` 并要求"同时显示"

---

## 字段填写指南

| 参数 | 是否必填 | 默认 |
|------|---------|------|
| 文章正文 | 必填 | —— |
| 风格预设 | 选填 | `academic-tech` |
| 风格描述 | 选填 | 无 |
| 输出路径 | 选填 | 返回到对话 |
| 目标平台 | 选填 | `bilibili, youtube` |
| 预估时长 | 选填 | 根据文字量估算（按 250 字/分钟） |
| 场景数量 | 选填 | 15-25（自动决定） |
