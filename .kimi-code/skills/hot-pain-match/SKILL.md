# Hot Pain Match — 热搜 × 痛点图谱匹配

将今日头条 + 微信实时热搜与 [40 岁中年危机痛点图谱](https://github.com/user-attachments) 进行交叉匹配，输出「最具内容创作价值」的热搜排行。

**为什么是头条 + 微信？**
- 今日头条用户画像：30-50岁男性为主，与「中年危机」目标受众完美重合
- 微信热文：高传播度的社交分发内容，反映中产/中年群体的真实关注点
- 两者互补——头条代表算法推荐（大众情绪），微信代表社交圈层（中产焦虑）

## 工作流

```
[Step 1] 获取热搜 — 并行拉取头条 + 微信双榜
    ↓
[Step 2] 合并去重 — 同一标题在双平台出现 = 自动提权
    ↓
[Step 3] 痛点匹配 — 每条热搜 vs 20 个痛点，计算匹配度
    ↓
[Step 4] 排序输出 — 按"创作价值分"排序，标注痛点 + 内容切入点建议
```

### 2.2 匹配算法

对每条热搜，遍历 20 个痛点，计算**创作价值分**：

```
创作价值分 = 关键词匹配度（0-10）× 痛点概率权重（0-1.0）× 话题讨论度加成（0.8-1.2）

其中：
- 关键词匹配度：热搜标题/描述中命中关键词数量 × 2（每个关键词 2 分，上限 10）
  - 最低要求：至少命中 2 个关键词（kw_score ≥ 4），过滤掉仅命中 1 个关键词的弱关联
- 痛点概率权重：直接使用痛点报告中的概率值（如 0.80, 0.52）
- 话题讨论度加成：综合计算
  - 🔥双平台热搜（头条+微信同时上榜）：强制 1.2
  - 单平台热度 ≥ 80万：1.2
  - 单平台热度 60-79万：1.1
  - 单平台热度 30-59万：1.0
  - 单平台热度 10-29万：0.9
  - 单平台热度 < 10万：0.8
```

**双平台加权说明**：同一条热搜同时出现在头条和微信，说明它同时触达了「算法推荐人群」和「社交传播人群」——穿透力更强，自动获得最高讨论度加成。

**匹配精度等级**：

| 匹配度 | 条件 | 示例 |
|--------|------|------|
| ⭐⭐⭐ 直接命中 | 标题含 3 个以上关键词 | "35岁程序员被裁员后……" → #1 就业歧视 |
| ⭐⭐ 高度相关 | 标题含 1-2 个关键词 | "某公司大规模优化" → #1 就业歧视 |
| ⭐ 间接关联 | 描述/背景含关键词 | "经济数据发布" → #3 财务脆弱（间接） |
| — 不相关 | 无任何关键词命中 | 纯娱乐八卦 → 不匹配任何痛点 |

### 2.3 匹配优先级规则

1. **高概率（>60%）× 直接命中** → 最高优先
2. **高概率（>60%）× 高度相关** → 第二优先
3. **中等概率 × 直接命中** → 第三优先
4. 纯娱乐/体育/八卦 → 默认不匹配（除非用户要求）

## 注意事项

- **娱乐八卦过滤**：默认不匹配纯娱乐/体育/明星八卦（除非与痛点明显相关，如明星离婚 → #16 离婚风险）
- **政治敏感回避**：涉及敏感政治话题的热搜谨慎处理，优先选择可安全讨论的社会/经济/生活话题
- **时效性优先**：创作价值分相同时，优先展示热度更高的热搜
- **双平台重叠是强信号**：头条+微信同时上榜的热搜，代表话题突破了圈层，是最佳创作素材
- **痛点图谱是锚点**：所有匹配必须基于痛点图谱中的 20 个痛点，不自行扩展
- **头条内容偏好**：头条用户偏好社会民生、家庭伦理、职场生存类话题——匹配这类的热搜天然具有更高创作价值
- **微信内容偏好**：微信热文偏向深度分析、中产焦虑、教育医疗类话题——匹配这类的热搜适合长文/观点类创作
